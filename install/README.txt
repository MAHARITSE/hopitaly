# Scripts d'installation HealthNet

- `install_all.bat` : Vérifie Python, installe Django 1.6.5, migre la base et lance le serveur.
- `run_server.bat` : Lance uniquement le serveur (`python manage.py runserver`).
- `setup_db.bat` : Exécute uniquement la migration (`python manage.py migrate`).

Utilisez `install_all.bat` pour une installation et un lancement rapide.
